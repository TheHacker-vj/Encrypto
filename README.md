# 🔐 Encrypto - My 12th Grade CS Project

Hey everyone! 👋 This is my Computer Science project from Class 12 - a simple app that lets you save secret messages (passwords, personal notes, whatever you want!) with basic encryption.

## ✨ What It Does

- Create an account with username & password
- Login to access your files
- Write secret messages and save them
- Open your saved messages (they get decrypted when you open them)
- Delete files you don't need anymore
- Sign out when you're done

## 🤔 Why I Made This

I wanted to build something useful while learning Python. Everyone has secrets they want to keep safe, right? So I thought - why not make a mini "vault" for text messages? It's not military-grade secure, but it taught me a lot about how encryption and file systems work!

## 🛠️ How It Works (Simple Version)

- Your password gets scrambled (encrypted) before saving
- Your messages also get scrambled when stored
- When you open a file, it unscrambles automatically
- All data lives in a CSV file (like a mini database)

## 📁 Files in This Project

- `encrypto.py` - The main code (everything runs from here)
- `APP.csv` - Gets created automatically when you sign up (stores all users and their encrypted messages)

## 🎮 How to Use

1. Download the code
2. Run it in Python (`encrypto.py`)
3. Choose Sign-in (if new user) or Login (if returning)
4. Follow the menu options to create/open/delete files
5. Type "4" to logout when done

## 💡 What I Learned

- How to read/write files in Python
- CSV files can work like a simple database
- Building menus that actually make sense to users
- Encryption is tricky but fun!
- How to handle errors so the app doesn't crash
